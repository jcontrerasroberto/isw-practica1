Primera práctica de ISW creada por Contreras Barrita José Roberto

Gestor de clientes creado a partir de las especificaciones del usuario.
Permite agregar, modificar, eliminar y listar clientes.

Estructura del proyecto:

Directorio docs: Contiene la documentación de este proyecto
Directorio res: Contiene los recursos para el funcionamiento del proyecto, en este caso el archivo para la base de datos
Directorio src: Código fuente programado en Python
Archivo run.sh: Script para correr el programado


Requisitos del sistema:

Python3 y git instalado en su maquina Linux.

Ejecución:

Clonar este repositorio
git clone https://github.com/jcontrerasroberto/isw-practica1.git

Entrar al directorio del proyecto
cd isw-practica1

Dar permisos al script run.sh
chmod +x run.sh

Ejecutar el programa
./run.sh

O con python3 directamente
python3 src/main.py

Para información más detallada leer docs/Documentacion.pdf
